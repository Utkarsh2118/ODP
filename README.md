*💸 Online Donation Platform*

An Online Donation Platform built using the MERN stack (MongoDB, Express, React, Node.js) that allows users to donate to various charities and causes in a secure and transparent manner.

## 🚀 Features

- 🔐 User Authentication (Register/Login using JWT)
- 🎯 Browse and view donation campaigns
- 💸 Donate to campaigns with multiple payment options
- 🧾 Automatic receipt generation for every donation
- 📊 Track total funds collected for each campaign
- 🛠️ Admin can create and manage campaigns

## 🧠 Tech Stack

- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT
- Payment Integration: (Planned: Razorpay)

## 🔄 Workflow

1. User registers or logs in
2. Views available campaigns
3. Selects a campaign and donates
4. Chooses a payment method
5. Receives a digital receipt after successful donation
